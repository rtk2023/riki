# 🤼 Sacensību kategoriju kalkulators

## Par projektu

**Sacensību kategoriju kalkulators** ir viegls, vienas lapas tīmekļa rīks, kas palīdz cīņas sporta dalībniekiem (un treneriem) ātri noteikt:

- piemēroto **svara kategoriju**,
- pareizo **vecuma grupu** (Jaunieši / Juniori / Pieaugušie),
- cik (kg) līdz **nākamajai kategorijai**.


### Izvades informācija

- **Aprēķinātais vecums** — automātiski no dzimšanas gada
-  **Vecuma grupa** — Jaunieši / Juniori / Pieaugušie
-  **Svara kategorija** — atbilstošā sacensību kategorija kg
-  **Atstatums** līdz nākamajai kategorijai
-  **Ieteikums** — vai sportists ir stabilā vai robežas kategorijā

### Validācija

Kalkulators pārbauda:
- vai aizpildīti visi lauki,
- vai svars ir reālistiskā diapazonā,
- vai dzimšanas gads ir derīgs,
- vai dzimums atbilst izvēlētajam cīņas veidam (sievietes nevar izvēlēties GR).
