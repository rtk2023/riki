Poké Team Helper.

Ievads:
Šis rīks ir paredzēts Pokémon spēlētājiem, lai palīdzētu plānot un optimizēt savu komandu sastāvu, nodrošinot ātru piekļuvi informācijai par Pokémonu tipiem un saderību.

Pārskats:
Projekta arhitektūra un struktūra:
Projekts ir izstrādāts, izmantojot HTML, CSS un JavaScript. Rīka galvenā loģika balstās uz klienta puses aprēķiniem.

Failu struktūra:
* `index.html` - Rīka galvenā struktūra un saskarne latviešu valodā.
* `style.css` - Vizuālais noformējums un responsīvā dizaina elementi.
* `script.js` - Galvenā loģika: datu apstrāde un komandas aprēķini.
* `README.md` - Projekta dokumentācija.

Iespējamie uzlabojumi:
* Integrācija ar publisko PokéAPI, lai iegūtu aktuālākos datus.
* Iespēja saglabāt izveidoto komandu pārlūkprogrammas lokālajā atmiņā (localStorage).
* Vizuāls grafiks, kas parāda komandas stiprās un vājās puses pret dažādiem tipiem.

Zināmie defekti:
Ir hard-coded saraksti, lai API atrastu pokemona vārdus, divu API nesaskaņas dēļ.
