# README
## Ievads
Šis projekts ir vienkāršs ūdens dzeršanas izsekošanas rīks, kas palīdz lietotājam sasniegt dienas mērķi.

## Pārskats
Projekts veidots kā viena HTML lapa ar iebūvētu CSS un JavaScript.

## Struktūra:
index.html – galvenais fails (UI + loģika)
CSS – dizains un animācijas
JavaScript – funkcionalitāte:
ūdens pievienošana
progresa aprēķins
localStorage datu saglabāšana
streak un achievements sistēma

## Iespējamie uzlabojumi
Lietotāja profils
Atgādinājumi dzert ūdeni
Grafiki/statistika
Tumšais režīms

## Zināmie defekti
Var ievadīt nereālus mērķus
Nav validācijas visiem gadījumiem
UI nav pilnībā responsive uz ļoti mazām ierīcēm
Nestrādā sasniegumi
