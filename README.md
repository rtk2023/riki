<h1>🧮 Ķermeņa tauku kalkulators</h1>

---

<h2>📌 Ievads</h2>

Šis rīks ir vienkāršs tīmekļa kalkulators, kas ļauj lietotājam aprēķināt aptuveno ķermeņa tauku procentu, izmantojot ievadītos datus.

---

<h2>📖 Pārskats</h2>

Projekts ir statiska tīmekļa lietotne, kas izstrādāta ar HTML, CSS un JavaScript.  
Visi aprēķini notiek lietotāja pārlūkā (bez backend).

<h3>🏗 Arhitektūra</h3>

```
Frontend  → HTML + CSS
Loģika    → JavaScript (script.js)
Backend   → nav izmantots
```

---

<h2>📁 Failu struktūra</h2>

```
/projekts
│
├── index.html      # Galvenā HTML lapa
├── style.css       # Dizains
├── script.js       # Aprēķinu loģika
└── README.md       # Dokumentācija
```

---

<h2>⚙️ Funkcionalitāte</h2>

```
[1] Lietotājs ievada:
    • dzimumu
    • svaru (kg)
    • augumu (cm)
    • vecumu

[2] Nospiež "Aprēķināt"

[3] Sistēma:
    → aprēķina BMI
    → aprēķina ķermeņa tauku %

[4] Rezultāts:
    → tiek attēlots uz ekrāna
```

---

<h2>🚀 Iespējamie uzlabojumi</h2>

```
+ Precīzāka formula (ar apkārtmēriem)
+ Rezultāta interpretācija (zems / normāls / augsts)
+ Responsīvs dizains mobilajām ierīcēm
+ Kļūdu validācija
+ Datu saglabāšana (localStorage)
+ Vizuāls rezultāts (krāsas / grafiks)
```

---

<h2>⚠️ Zināmie defekti</h2>

```
- Aprēķins ir aptuvens
- Nav pilnas ievades pārbaudes
- Nav kļūdu apstrādes sarežģītiem gadījumiem
```
